# Web前端工具箱

> 建立一个自己的工具库

### 演示地址
#### [fe-tool](http://tool.chenzejiang.com)

## 如何使用?

``` bash
# 安装依赖
npm install

# 本地调试预览 localhost:8080
npm run dev

# 打包发布
npm run build
```

> 技术栈
* vue2
* vue-router
* less
* es6

> 打包
* webpack

> 插件
* jquery
* qrcode
* colpick
